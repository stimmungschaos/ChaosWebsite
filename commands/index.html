ChaoslyBot Commands

-afk
Stellt dich halt AFK. Es gibt auch noch -gn oder so, was einfach Abwandlungen sind.
-wetterbericht
Gibt dir den Wetterbericht von einer beliebigen Stadt aus.
-join
Damit joint der Bot einem Channel.
-part
Mit diesem Command leavet der Bot deinem Channel.
-uhrzeit
Gibt dir die aktuelle Uhrzeit aus.
-txtupload
Uploadet einen Text deiner Wahl an Lucas' Paste.
